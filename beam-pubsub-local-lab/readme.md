A beginner‑friendly, zero‑cost setup to learn streaming on your laptop—without touching real GCP.

This repo shows how to:

✅ Create a Pub/Sub topic (locally)

✅ Create a Pub/Sub subscription (locally)

✅ Publish one test message (locally)

✅ Run a minimal Apache Beam streaming pipeline


Prerequisites

✅ Java 11+ (required by the Pub/Sub emulator)

✅ Google Cloud SDK installed (for gcloud + Pub/Sub emulator)

✅ Python 3.9+ recommended
